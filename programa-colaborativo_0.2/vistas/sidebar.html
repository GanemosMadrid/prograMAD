¿Para qué sirve esta web?

Esta web sirve para elaborar de manera colectiva el programa que Ganemos Zaragoza presentará el próximo mes de mayo a las elecciones municipales.

Cualquier persona que se registre puede presentar aquí su propuesta, así cómo debatir, valorar comentar y sugerir mejoras a aquellas propuestas publicadas.

Esto significa que este foro es únicamente para debatir sobre programa. Consulta la web de Ganemos Zaragoza para ver otros procesos de participación online abiertos.
