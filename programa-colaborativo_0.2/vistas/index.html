Programa colaborativo de Ganemos Zaragoza

En Ganemos Zaragoza hemos puesto en marcha un proceso para elaborar de forma colectiva un programa electoral con las necesidades y demandas de los habitantes de la ciudad.

El primer paso de este proceso es identificar cuáles son aquellas propuestas sobre las que existen un amplio consenso ciudadano. A través de esta web podemos recoger y evaluar de forma colaborativa cuál es el grado de acuerdo sobre diferentes propuestas. Las de más consenso sobre cada área se tomarán como la base sobre la que construir el programa de Ganemos Zaragoza.
